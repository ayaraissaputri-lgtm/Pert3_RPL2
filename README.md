Proyek ini dibuat menggunakan NetBeans dan berfungsi sebagai aplikasi sederhana untuk mengelola data melalui antarmuka input yang terhubung dengan database PHP. Pengguna dapat memasukkan data melalui form yang tersedia, kemudian menyimpannya ke dalam database dengan tombol Simpan. Selain itu, aplikasi ini menyediakan tombol Hapus untuk menghapus data yang dipilih, serta tombol Refresh untuk memperbarui tampilan data agar selalu menampilkan informasi terbaru dari database. Dengan demikian, proyek ini memudahkan proses manajemen data secara interaktif dan efisien.
